# 數學小課堂（教育示範原型）

這是一個針對國小低年級學生的「遊戲式數學教學」靜態網站原型，包含首頁、影片教學選單、遊戲選單與隨堂小考（題庫、隨機抽題、即時回饋與評分）。

主要檔案：
- `index.html`：主頁與彈窗結構
- `style.css`：樣式（童趣、繽紛但不刺眼）
- `sketch.js`：主要互動邏輯（彈窗、題庫與小考流程）

如何本機開啟：
1. 直接用瀏覽器打開 `index.html`（在檔案總管上雙擊或以瀏覽器開啟）。
2. 若要在本機啟動簡易 HTTP 伺服器（避免某些瀏覽器在 file:// 下的限制），可使用以下指令：

Windows PowerShell 範例：

```powershell
# 進入專案資料夾
cd 'C:\Users\USER\Downloads\20251217'
# 使用 Python 3 的簡易伺服器（若已安裝 Python）
python -m http.server 8000
# 然後在瀏覽器開啟 http://localhost:8000
```

功能與注意事項：
- 「影片教學」與「開始遊玩」目前為選單與占位（影片或遊戲邏輯請後續提供），按下選項會顯示占位訊息。
- 「隨堂小考」：系統會從內部題庫（最多 20 題）隨機抽 5 題。題型可選：綜合/加法/減法/乘法。每題提交後會立即給予回饋（答對/答案提示），最後顯示分數與鼓勵訊息。

可擴充項目（建議）：
- 上傳影片資源並在「影片教學」中嵌入播放元件（或使用 YouTube 連結）。
- 為「開始遊玩」實作互動遊戲（可用 canvas 或 p5.js）。
- 將題庫改為可由老師管理的 JSON 檔或後端 API，支援題型與難度分級。
